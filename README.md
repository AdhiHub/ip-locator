## 🔍 How to Find Someone's IP

This tool looks up IPs — it doesn't find them. But here are common ways people get IPs:

| Method | How it works |
|--------|-------------|
| **Your own IP** | Leave input blank — this tool shows it automatically |
| **Website logs** | Any website you visit sees your IP (server logs) |
| **Discord** | Right-click a call → "Copy IP" (during direct calls) |
| **Grabify / IP loggers** | Create a tracking link → send it → they click → you get their IP |
| **Game servers** | Some games show connected players' IPs in server logs |
| **Email headers** | Raw email headers contain the sender's IP |

> ⚠️ **Ethical use only.** Tracking someone's IP without consent can be illegal in many countries. Use only on your own devices or with explicit permission.
